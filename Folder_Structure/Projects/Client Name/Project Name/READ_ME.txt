THE PROJECTS ARE DIVIDED INTO THE FOLLOWING FOLDERS.

- DESIGN
- RELEASED
- WORK


IN THE DESIGN FOLDER YOU CAN ADD ALL THE PROJECT YOU ARE CURRENTLY WORKING ON. THEFILES IN THIS FOLDERS CAN BE MODIFIED.

IN THE RELEASED FOLDER; YOU ARE ONLY GOING TO PLACE THE POJECTS THAT HAVE BEEN RELEASED, OR MANUFACTURED. YOU ARE NOT ALLOWED TO MODIFY THIS FILES FOR ANY REASON!

IN THE WORK FOLDER YOU CAN ADD ALL THE FILES RELEVANT FOR WORKING WITH THE PROJECT. YOU CAN ADD OR DELTE FILES AS YOU GO.
